# OIBSIP Task 1 - Iris Flower Classification

This project is part of the Oasis Infobyte Internship Program.

## Objective
To classify Iris flower species (Setosa, Versicolor, Virginica) using machine learning.

## Dataset
Iris dataset with sepal and petal measurements.

## Algorithm Used
Logistic Regression

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Output
The model successfully classifies the iris flower species with good accuracy.
